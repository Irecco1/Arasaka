# Arasaka
Big work in progress project! Don't expect anything usable in near future. Project start: 17.03.2026

Arasaka is a project to introduce economy to minecraft servers using CC: Tweaked and Advanced Peripherals.

It's made with heavy modded servers in mind, like All The Mods 10, where you can't use any item as a currency, because everything can be farmed on a massive scale.

It allows you to setup a "server", which will be responsible to take care of "clients" requests, allowing you to sell items, buy items, run a casino, stock market, minigames or anything else. I am trying to keep the code as simple as possible, so if you want to add something yourself, it shouldn't be the hardest thing if you know how to program. I also happily accept ideas on what to add, so you can let me know your idea and I might add it to the program.

# How does it work?
To make it work you will need:
- CC: Tweaked PC that will have to be connected to your AE2 system to automatically sell items to other players. You will be able to set what you want to sell and how expensive it should be.
- Some peripherals from Advanced Peripherals (exact details will be available on launch)
- Portable PC from CC: Tweaked with installed software that you will give (or sell) to players

Each player that want to use this system will have to do a few things:
- Register in the program using his own nickname. Each player can have only one account
- Give you a linked memory card. This is huge because with this, you get full access to their entire inventory all the time, but that's why the program is called "Arasaka" and not something weak like "We respect consumers" ;)

That's it. After they register, you insert their memory card to Inventory Manager and refresh the peripherals from your admin PC, everything should be ready to go!
